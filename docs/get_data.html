get_data¶

get_data.fetch_historical_data(crypto_id, start_date, end_date, api_key=None)¶

Fetch historical hourly cryptocurrency data from an external API for a specific time range with support for pagination.

Parameters:

  • crypto_id – The cryptocurrency identifier (e.g., ‘BTC’, ‘ETH’).

  • start_date – The start date (inclusive) as a string in ‘yyyy-mm-dd HH:MM:SS’ format.

  • end_date – The end date (inclusive) as a string in ‘yyyy-mm-dd HH:MM:SS’ format.

  • api_key – API key for authentication (optional).

Returns:

A pandas DataFrame with columns [‘date’, ‘price’] and a DatetimeIndex (naive, hourly frequency).

get_data.fetch_specific_historical_hours(crypto_id, hours_list, api_key=None)¶

Fetch specific hourly cryptocurrency data for a list of timestamps from an external API.

Parameters:

  • crypto_id – The cryptocurrency identifier (e.g., ‘BTC’, ‘ETH’).

  • hours_list – A list of timestamps (datetime objects) for which to fetch data.

  • api_key – API key for authentication (optional).

Returns:

A pandas DataFrame with columns [‘date’, ‘price’] containing the fetched data.
