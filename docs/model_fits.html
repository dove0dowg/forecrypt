model_fits¶

model_fits.fit_arima_model(df, **kwargs)¶

arima model training function. all parameters must come from kwargs[‘specific_parameters’] (e.g. order, seasonal_order).

model_fits.fit_ets_model(df, **kwargs)¶

ets model training function. all parameters must come from kwargs[‘specific_parameters’] (e.g., trend, seasonal, seasonal_periods).

model_fits.fit_theta_model(df, **kwargs)¶

theta model training function. parameters come from kwargs[‘specific_parameters’] if needed (like m, method, etc.).
